# Power BI Analysis: DMV Trend Battle
This is my project for the **New Force Cohort 12**.

## Project Overview
The goal for this project was to analyze the search trends and data patterns in the DMV area regarding two popular brands.

##Data Source
- **Source**: Conducted Raw search data and downloaded from **Google Trends**.
- **Format**: The data was originally a **CSV file** and was later converted to an **Excel (.xlsx)** workbook (`DMV_Trend_Battle.xlsx`) for cleaning and preparation to upload to **Power Bi.**

## Steps Completed:
- **Data Preparation**: I used **Excel** to clean the raw CSV data retrieved from **Google Trends** and format it into a structured table.
- **Visual Analysis**: I uploaded the cleaned data to the **Power BI Service** (website version) to help build my interactive dashboard.
- **Dashboard Development**: I created a report (`DC Trends HH vs TNF.pbix`) that visualizes key performance indicators and trends over a 10 year time span.
- **Version Control**: I used my **MacBook Terminal** to initialize a Git repository and push my final project files to this GitHub account.

## Files Included:
- `DC Trends HH vs TNF.pbix` - This is my final Power BI Dashboard file.
- `DMV_Trend_Battle.xlsx` - This is the Source used to convert into a Excel data file used for the analysis.
